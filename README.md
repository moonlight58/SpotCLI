# findSpot

C script that would help me find spotify music and add it to my liked songs playlist without having to open spotify app or open.spotify.com

# Usages
```bash
findSpot <type_of_search> "thing_you_search"


<type_of_search> :
- albums -> search for albums
- artists -> search for artists
- audiobooks -> search for albums
- player -> will be used to allow play preview of musics
- playlists -> search for playlists
- track -> search for music(s) 
- users -> search for users

"thing_you_search" :
the context you write in it will be used for the search
```
